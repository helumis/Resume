# Resume
## 自傳
  大學期間修習生物統計與資料科學學程，熟悉以 R 與 Python 進行資料清理、探索與模型建構，並參與從前處理到推論分析與報告呈現的完整流程。多次專題與實作中，主要負責釐清問題並穩定推進任務，確保成果具備可靠性與解釋力。

  在「2025資料科學漫步」競賽中，以混合效應模型分析不完全追蹤資料，並控制個體差異與時間效果後比較疫情前後變化。結果顯示，疫情後青少年平均幸福感僅輕微下降，疫情本身未呈現直接顯著影響。進一步發現，「家長對網路使用的管制程度」與幸福感呈顯著正向關聯，顯示家庭數位行為管理可能比疫情因素更具解釋力；且該管制比例於疫情期間上升、疫情後下降，反映其受外部環境（如遠距學習需求）影響而動態變化。此結果也使我理解，資料分析不僅在於顯著性判定，更在於將變數關係置於情境中解釋，避免過度簡化因果推論。
